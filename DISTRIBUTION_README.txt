========================================
  PoE Goblin - Path of Exile 2 価格チェックツール
========================================

## 初回セットアップ（重要！）

アプリを使用する前に、.envファイルを作成する必要があります：

方法1: env.templateをリネーム（簡単）
1. アプリと同じフォルダにある `env.template` を `.env` にリネーム
2. メモ帳で開く
3. your-email@example.com を自分のメールアドレスに変更
4. 保存

方法2: 新規作成
1. メモ帳で新規ファイルを作成
2. 以下を記入： API_EMAIL=your-email@example.com
3. your-email@example.com を自分のメールアドレスに変更
4. アプリと同じフォルダに `.env` として保存

※ このメールアドレスは価格情報APIへのアクセスに使用されます

## 使い方

1. アプリを起動（PoE-Goblin-x.x.x-portable.exe をダブルクリック）
2. Path of Exile 2 でアイテムにカーソルを合わせる
3. Ctrl+C でアイテム情報をコピー
4. Ctrl+G で価格表示ウィンドウを開く
5. Esc または画面外クリックでウィンドウを閉じる

## ホットキーの変更

アプリ内の「設定」ボタンからホットキーをカスタマイズできます。

## システム要件

- Windows 10/11 (64bit)
- インターネット接続（価格情報の取得に必要）

## トラブルシューティング

### Windows Defenderの警告が出る場合

1. 「詳細情報」をクリック
2. 「実行」をクリック
   ※ このアプリはコード署名されていないため、警告が表示されます

### 価格が表示されない場合

- アイテムが正しくコピーされているか確認してください
- インターネット接続を確認してください
- 対応していないアイテムの可能性があります
